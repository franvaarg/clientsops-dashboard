# ClientsOps — Customer Operations SaaS Dashboard

Modern SaaS dashboard built for customer success and operations teams.
ClientsOps provides a centralized interface to monitor customer accounts, revenue growth, support activity and account health.

This project was developed as a **front-end portfolio application** to demonstrate modern dashboard architecture used in enterprise SaaS platforms.

## Installation

Clone the repository:

```
git clone https://github.com/franvaarg/clientsops-dashboard.git
```

Navigate to the project folder:

```
cd clientsops-dashboard
```

Install dependencies:

```
npm install
```

Start the development server:

```
npm run dev
```

---

## Build

To generate a production build:

```
npm run build
```
